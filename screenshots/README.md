# Screenshots

This folder will contain screenshots related to my cybersecurity home lab.

Screenshots may include examples of dashboards, monitoring views, service layouts, and other lab-related visuals. Before uploading any screenshot, sensitive information should be reviewed and removed.

## Screenshot Review Checklist

Before committing screenshots to GitHub, I will check for:

* Passwords
* Tokens or API keys
* Private IP addresses
* Hostnames
* Usernames
* Email addresses
* Internal URLs
* Personal file paths
* Device names that should not be public
* Employer-owned or confidential information

## Notes

Screenshots should be sanitized before being uploaded. If a screenshot contains sensitive information, it should be edited, blurred, cropped, or excluded from the repository.
